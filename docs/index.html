Easy Installation

1

Download

Click the download button above to get the latest version of PureFeed

2

Extract

Unzip the downloaded file to any folder on your computer

3

Open Extensions

Go to your browser's extensions page:
• Chrome: chrome://extensions/
• Edge: edge://extensions/
• Brave: brave://extensions/

4

Install

Enable "Developer mode" (toggle in top right), then click "Load unpacked" and select the extracted folder
